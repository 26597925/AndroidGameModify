# 快捷键
```
F12 --> 暂停程序
F9 --> run 
F8 --> step over 
F7 --> step into 
F5 --> CPU 窗口
F4 --> 运行到当前选中行
F3 --> ADB Shell 	Alt F3 --> GDB Shell 


ART Debug/View/ELF Data 以及 ART Debug/View/ELF Code
都可以解析Android下的so和bin文件
两者的显示方式不同Data是以原属二进制为主，Code是以汇编代码和ELF结构体代码为主。
```
